# Контекст Проекта: $LUDO Web3 Casino (Solana)

## 1. Обзор проекта
Роскошное Web3-казино на блокчейне Solana.
**Тикер токена:** $LUDO
**Визуальный стиль / Дизайн-код:** AAA-уровень, Web3 Luxury (стеклянный морфизм). Основные элементы UI (слоты, кнопки, текст) используют Золото (`#FFD700`) и Красный (`#FF2A4D`) для создания классической "дофаминовой" атмосферы казино (азарт, богатство, сжигание). Фон и эмбиент-свечение используют фирменные цвета Solana — Фиолетовый (`#9945FF`) и Зеленый (`#14F195`), чтобы проект четко ассоциировался с экосистемой.

## 2. Текущий стек технологий
- **Front-end прототип:** Чистый HTML5, CSS3, ES6 JavaScript.
- Фреймворки пока не используются. Проект готов ко **Фазе 2** (перенос на React/Next.js).

## 3. Реализованные фичи в демо-прототипе
1. **Интерфейс**: Адаптивный Glassmorphism, атмосферные плавающие орбы (ambient orbs), сетка на фоне.
2. **Игровой автомат (Слоты)**: Анимация вращения барабанов с помощью `animateReel`, набор символов (`['🪙', '🍒', '7️⃣', '💎', '🚀']`), таблица множителей. Имитация выигрышей и проигрышей.
3. **Аудио-движок**: Синтезированные звуки (Web Audio API) для кликов, спинов, победы (мажорные аккорды) и проигрыша (диссонанс).
4. **Интеграция маскота (`LUDO.png`)**: Увеличенный маскот располагается справа от автомата. Он левитирует (idle) и динамически реагирует на игру через текстовые облачка (без эмодзи: "Let's gooo!", "JACKPOT! We did it!"), а также прыгает при победе и трясется при проигрыше.
5. **Система Авто-ставок (Auto-Bet)**: Блок `AUTO SPINS` с настройкой лимита. Логика автоматически крутит слоты заданное количество раз, пока есть баланс.
6. **Лидерборд (Leaderboard)**: Расположен слева. Показывает топ "моковых" выигрышей дня (усеченные кошельки + суммы в $LUDO).
7. **Социальные сети**: Круглые glass-кнопки (X, Telegram, DexScreener) зафиксированы в правом нижнем углу.
8. **Подключение кошелька (Mock)**: Имитация подключения Phantom, смена стейта кнопок с "LOCKED" на "SPIN".
9. **Live Feed и сжигание (Burn)**: Имитация логов транзакций в футере, где показывается сжигание или выигрыши других игроков.
